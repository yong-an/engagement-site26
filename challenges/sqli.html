NOTE: WHAT IS SQL INJECTION?

SQL injection works by breaking out of the string context in the SQL query. If the username field is directly embedded in the query, typing a single quote ' can close the string early, letting you inject your own SQL logic.

NOTE: HOW TO BYPASS LOGIN

Try making the WHERE clause always evaluate to TRUE. A classic trick: enter ' OR '1'='1 as the username. This turns the query into WHERE username='' OR '1'='1' — which is always true, bypassing the password check entirely.
