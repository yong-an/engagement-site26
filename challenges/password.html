NOTE: WHAT IS A DICTIONARY ATTACK?

Instead of trying every combination, a dictionary attack tests a list of common passwords. Each is hashed and compared against the leaked value. If the user chose a weak password, it will match.

NOTE: WHAT HASH ALGORITHM?

The hashes are MD5, fast to compute and cryptographically broken. Modern systems use bcrypt, scrypt, or Argon2, which are intentionally slow and salted to resist this exact attack.
