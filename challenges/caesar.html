Your packet sniffer intercepted an encrypted radio transmission They are using a Caesar cipher.

Use the decoder below to try different shift values. Find the one that produces readable English, then extract the hidden password from the message.

NOTE: WHAT IS A CAESAR CIPHER?

Each letter is shifted by a fixed number. With shift=3: A→D, B→E, Z→C. To decode, shift the other way. There are only 25 possible shifts — try them all until you read English.

NOTE: WHAT SHIFT TO TRY?

The message mentions the shift itself. One line says IXYVJ. The word "shift" matches this code. Try decoding that word and count the letters to find the shift value.
